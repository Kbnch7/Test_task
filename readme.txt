Для использования запустите main.py, вы получите краткую инструкцию по использованию.
Для добавления книги воспользуйтесь командой <add *Название книги в двойных кавычках* *Автор* *Год выпуска*>
Для удаления книги воспользуйтесь командой <delete *id книги*>
Для поиска книги воспользуйтесь командой <find *фильтр поиска(title/author/year)*=*значение фильтра*> (если указываете title, то значение нужно писать в кавычках)
Для отображения всех книг воспользуйтесь командой <show_all>
Для изменения статуса книги воспользуйтесь командой <change_status *id книги* *новый статус (avalible/unavalible)*>
Для сохранения изменений воспользуйтесь командой <save>
Для выхода из программы воспользуйтесь командой <quit>
